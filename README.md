#Car Price Prediction using Machine Learning

##  Overview
This project predicts the price of cars using Machine Learning techniques.  
A **Linear Regression model** is trained on a dataset containing different car features such as manufacturer, model, engine size, and other specifications to estimate the car price.

The project demonstrates the complete **machine learning workflow**, including data preprocessing, model training, prediction, and evaluation.

## 📊 Dataset
The dataset used in this project is the **Cars93 dataset**.

### Features in the Dataset
- Manufacturer  
- Model  
- Type  
- Engine Size  
- Horsepower  
- MPG (City / Highway)  
- Weight  
- Wheelbase  
- Length  
- Width  
- Price (Target Variable)

## 🛠 Technologies Used
- Python  
- Jupyter Notebook  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

## ⚙️ Machine Learning Workflow
The following steps were performed in this project:

1. Data Collection  
2. Data Cleaning  
3. Handling Missing Values  
4. Converting Categorical Data into Numeric  
5. Splitting Dataset into Training and Testing Sets  
6. Training the Machine Learning Model  
7. Predicting Car Prices  
8. Evaluating Model Performance  

## 🤖 Model Used
- Linear Regression

## 📈 Model Evaluation Metrics
- Mean Absolute Error (MAE)  
- Mean Squared Error (MSE)  
- R² Score  

## 📌 Conclusion
This project demonstrates how machine learning can be used to predict car prices based on vehicle specifications. It provides practical experience with **data preprocessing, regression models, and model evaluation**.
